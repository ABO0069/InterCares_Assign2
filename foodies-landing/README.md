# Foodies Landing Page

## Overview
This project is a landing page for an organic food store called "Foodies". It is built using HTML, CSS, Tailwind CSS, and Bootstrap. The landing page showcases the store's offerings and provides a user-friendly interface for visitors.

## Project Structure
```
foodies-landing
├── src
│   ├── index.html          # Main HTML document for the landing page
│   ├── styles.css          # Custom CSS styles
│   ├── tailwind.css        # Compiled Tailwind CSS file
│   ├── scripts
│   │   └── main.js         # JavaScript for interactivity
│   └── components
│       ├── header.html     # Header section
│       ├── hero.html       # Hero section
│       ├── features.html    # Features section
│       └── footer.html     # Footer section
├── assets
│   ├── fonts               # Directory for font files
│   └── svgs                # Directory for SVG files
├── package.json            # npm configuration file
├── tailwind.config.js      # Tailwind CSS configuration
├── postcss.config.js       # PostCSS configuration
├── .gitignore              # Git ignore file
└── README.md               # Project documentation
```

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd foodies-landing
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Build the Tailwind CSS:
   ```
   npm run build
   ```
5. Open `src/index.html` in your browser to view the landing page.

## Features
- Responsive design using Tailwind CSS and Bootstrap.
- Interactive elements powered by JavaScript.
- Modular components for easy maintenance and updates.

## Usage
- Customize the content in the `src/components` directory to update the landing page sections.
- Modify styles in `src/styles.css` for any additional customizations.
- Use the `assets` directory to add or update fonts and SVGs.
